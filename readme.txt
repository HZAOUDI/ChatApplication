Database's name is: chat_db

We're using MySQL for storing data
Here is a guide how to restore the database:
- Open your MySQL Workbench
- In menu bar, click to icon "Create a new schema..."
- In name box, type: "chat_db"
- Click Apply
- Now you have your new database name "chat_db". Double click to that database in schema section.
- Now paste the code from the chat_db.sql file and run it!
- Done!

How to run:

- You should open this project using Netbeans IDE (not Eclipse) by importing the project in .zip format or
just click on open project.
- Firstly, run server: ServerFrame.java and launch it from the button "Lancer serveur" ! 
- Then, run ClientFrame.java. Each user will run one ClientFrame
